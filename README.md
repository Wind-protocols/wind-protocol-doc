# Website

This website is built using [Docusaurus 2](https://v2.docusaurus.io/), a modern static website generator.

## Installation

```console
yarn install
```

## Run serve

```console
npx browserslist@latest --update-db 

yarn upgrade @docusaurus/core@latest @docusaurus/preset-classic@latest

yarn run build

yarn run serve
```
